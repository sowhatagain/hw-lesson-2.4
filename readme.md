#Задание:

Определите модель с наилучшим показателем метрики accuracy, используя различные модели и варьируя их параметры.
dataset load_digits

#Решение:

Digits KNN and Tree.ipynb - для KNN и Дерева
Digits Logistic Regression.ipynb - для логистической регрессии

#Ответ, рассуждения:

Для оценки наилучшей модели определили суммарную площадь под кривыми AUC-ROC по всем классам (подробнее см. ноутбуки).
* Дерево:  9.9417451497712221
* KNN:   9.1407308353424064
* Лог регрессия:  9.9523263606161141

Все модели с подобранными параметрами показали очень высокую точность предсказания класса.
С небольшим отрывом от KNN лучше себя показала логистическая регрессия; скорее всего даже улучшит свой результат, если использовать более сложные встроенные алгоритмы.
Однако, с регрессий нужно обращаться осторожно, чтобы не получить высокой регуляризации.
